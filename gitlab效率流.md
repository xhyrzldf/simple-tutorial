## Git知识的基础整理

### 常用命令

| means | command |
| ---- | ---- |
| git init | 初始化git仓库 |
| git clone | 克隆 |
| git add . | 添加到暂存区 |
| git status | 查看文件提交状态 |
| git commit -m | 提交 |
| git remote add `[repo name]` `[repo location]` | 增加远程仓库 |
| git remote -v | 查看远程仓库 |
| git remote rm `repo name` | 移除远程仓库 |
| git push `optional [repo name] [repo branch]` | push到远程仓库 |
| git pull | pull到远程仓库 |
